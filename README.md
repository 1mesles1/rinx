# rink

Консольная читалка FB2 файлов на Rust с использованием Ratatui.


Это просто проект для изучения Rust. Ели запускается - уже хорошо.

### Особенности:
- Выравнивание текста по ширине (Justify).
- Поддержка Unicode (кириллица).
- Оглавление и информация о книге.
- Автоматическое сохранение позиции чтения.

### Запуск:
`cargo run -- путь_к_файлу.fb2`
